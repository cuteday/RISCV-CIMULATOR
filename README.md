# RISCV-CIMULATOR
C++ Implementation of a simple risc-v simulator.

## Usage
~~~
    make
    cd build
    ./run [args]
~~~

## Arguments
~~~
-f [risc-v64 file]: run executable in simulator
-d: print debug impormation for simulator
-e: print elf file parsing results
-cache: enable cache

-h: help information

~~~
